🍎 Apple MacBook 3D Website Clone

A high-fidelity clone of Apple's MacBook landing page featuring smooth animations, scroll-based storytelling, and 3D-like interactions built using React.js and GSAP.

🔗 Live Demo: https://your-vercel-link.vercel.app/

📌 Overview

This project replicates the premium feel of Apple’s website by implementing advanced frontend techniques such as:

Scroll-triggered animations
Timeline-based transitions
Component-driven UI architecture

It focuses on delivering a smooth, interactive, and visually rich user experience.

✨ Features
🎯 Smooth scroll animations using GSAP ScrollTrigger
💻 3D-like MacBook interaction effects
⚡ Optimized performance with React
🎨 Clean and modern Apple-inspired UI
📱 Fully responsive design
🔄 Reusable and modular components
🛠️ Tech Stack
React.js – Frontend framework
GSAP – Animations and transitions
ScrollTrigger – Scroll-based animation control
CSS / Tailwind – Styling
Vercel – Deployment

⚙️ Installation & Setup
1️⃣ Clone the repository

git clone https://github.com/your-username/macbook-3d-clone.git

2️⃣ Navigate into the project

cd macbook-3d-clone

3️⃣ Install dependencies

npm install

4️⃣ Start development server

npm start
